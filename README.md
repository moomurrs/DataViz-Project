# DataViz-Project
Visualizing America's Energy Transformation through graphs and interactive visualizations.

## Local Setup
1. `uv sync`
2. `./.venv/Script/Activate`
3. Open Jupyter on [DataViz_Project.ipynb](DataViz_Project.ipynb)
